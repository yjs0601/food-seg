# 🤖 병렬로봇 기반 음식 이물질 자동 분리 시스템

> YOLOv11 세그멘테이션 모델을 활용하여 병렬로봇이 음식과 이물질을 실시간으로 인식하고 자동으로 분리하는 시스템

---

## 📌 프로젝트 개요

식품 제조 공정에서 발생하는 이물질 혼입 문제를 해결하기 위해, 컴퓨터 비전 기반의 자동 분리 시스템을 개발했습니다.  
YOLOv11 Instance Segmentation 모델로 음식과 이물질을 픽셀 단위로 구분하고, 병렬로봇이 이물질을 실시간으로 제거합니다.

---

## 🎯 주요 기능

- **실시간 객체 인식**: YOLOv11을 활용한 음식 / 이물질 실시간 세그멘테이션
- **직접 데이터 수집 및 라벨링**: 660장의 커스텀 데이터셋 구축
- **높은 인식 정확도**: 약 98% 정확도 달성
- **로봇 제어 연동**: 인식 결과를 병렬로봇 제어 신호로 변환

---
